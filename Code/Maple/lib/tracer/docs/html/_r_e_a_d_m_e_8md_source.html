14 # About the C wrapper
15 
16 Tracer is written in C++11, but has an integrated C wrapper. All (supported) C++ class objects can be acquired with
17 `tr_get<ClassName>` functions and must be destroyed with `tr_free<ClassName>` functions.
18 
19 All (supported) methodes can then be called with `tr_<ClassName>__<functionName>`.
20 
21 The doxygen generated documentation can be found here: https://mensinda.github.io/tracer/html/tracer_8h.html

44 ## More advanced setup
45 
46 With this setup it is possible to customize the output and signal handler.
47 
48 ```cpp
49 auto *tHandler = TracerHandler::getTracer(); // Get the singleton
50 auto cfg = tHandler->getConfig(); // Get the current config
51 // Edit cfg
52 
53 tHandler->setConfig(cfg); // Update the config
54 auto printer = PrinterContainer::fancy(); // Generates a printer
55 // Edit printer config
56 
57 tHandler->setup(std::move(printer)); // Sets things up. Now the signal handler is setup
58 ```
59 
60 or using C:
61 
62 ```c
63 tr_TracerHandler_t * trTH = tr_getTracerHandler(); // Get the handler
64 tr_TracerHandler_Config_t cfg = tr_TracerHandler__getConfig(trTH); // get the current config
65 
66 // edit cfg
67 tr_TracerHandler__setConfig(trTH, cfg); // Update the config
68 
69 tr_Printer_t *systemPrinter = tr_getPrinter__system(); // Generates a printer
70 // Edit printer config
71 tr_TracerHandler__setup(trTH, systemPrinter); // Sets things up. Now the signal handler is setup
72 
73 // Do some cleanup
74 tr_freePrinter(systemPrinter);
75 tr_freeTracerHandler(trTH);
76 ```
